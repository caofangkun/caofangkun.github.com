Ssh免密码登录

Dec 26th, 2012 | Comments

1 mkdir .ssh

2 cd .ssh

sudo chmod 700 . //这一步很重要

3 ssh-keygen -t rsa

4 cat rsa_d.pub >> authorized_keys

当然也可以： cp rsa_d.pub authorized_keys

5 chmod 644 authorized_keys //这一步很重要
